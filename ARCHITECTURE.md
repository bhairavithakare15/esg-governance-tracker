# ESG Governance Tracker - System Architecture

## Overview

The ESG Governance Tracker is a full-stack web application for managing ESG (Environmental, Social, Governance) assessments and tracking organizational performance.

## Technology Stack

| Component | Technology |
|-----------|-----------|
| Frontend | React 18 + Recharts |
| Backend | Node.js + Express.js |
| Database | PostgreSQL |
| Authentication | JWT |
| Deployment | Docker + Docker Compose |

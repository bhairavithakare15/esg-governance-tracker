# ESG Governance Tracker

Production-grade ESG (Environmental, Social, Governance) analysis system.

## Features

- **Dashboard** - Real-time ESG score overview
- **ESG Criteria** - 6 key criteria with weights
- **Score Input** - Weighted score calculation
- **Gap Analysis** - Gap from ideal score
- **Reports** - AI-like recommendations
- **Settings** - Organization management

## Installation
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
```
